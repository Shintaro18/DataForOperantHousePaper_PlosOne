[Name]
AnalysisForActualDelay_Task6(DNMTP)

[Usage]
This program calculate actual delay time in the DNMTP task of the Operant House.

[Requirement]
Windows OS (including 10/11)

[How to use]
1 Run "ActualDelayCalc.exe"
2 Drag & drop ***Task6_Touch.csv files onto the window.
3 When the analysis is completed, the window will close and a file ***_AveActualDelay.csv will be created in the folder where data files exist.
